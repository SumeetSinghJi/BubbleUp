Detailed Description

A Base class for initialisating SDL button subclasses e.g. push buttons, sliders, text fields. 

                                SDL BUTTON BASE CLASS

DESCRIPTION:

The C++ language SDL Framework code below can be implemented in any SDL derived project as a method to create button object instances of many variations such as radio buttons, volume slider buttons, Input text fields and submit buttons.

The steps below outline the method to initialise, add to a vector, and handle the objects.

Button navigation includes multiple variations from shortest path key direction algorithms to find nearest button using directional keyboard/gamepad movement to clicking and even dragging buttons.

REQUIREMENTS

The below libraries are required to instantiate the buttons

  • SDL2
  • SDL2_ttf
  • SDL2_image

find nearest button

uses find_euclidean_distance() to measure Euclidean distance used in finding closest button to navigate to using directional navigation keys E.g. if you press up directional arrow key on a keyboard or gamepad all the buttons north in -y scale will be calculated to find the nearest button. then sets the closet button to isSelected = true
