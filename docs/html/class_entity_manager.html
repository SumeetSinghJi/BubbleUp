Detailed Description

Entity Manager manages the initialisation of entites from Entity.hpp.

Utility class that works alongside any entity class to generate entities for your game/project. e.g. procedural generate entities for a level or use with a level editor button select to create custom maps with specific entities.

EXAMPLE

  1. Include the libs header

include "../headers/EntityManager.hpp"

  1. You can create new Entitites for your game by defining their characteristics here and adding it to Entity.hpp
  2. Then to initialise them randomly in your scene call EntityManager::random_procedural_generation(renderer, entities, SCREEN_WIDTH, SCREEN_HEIGHT, 10, 10, 10);
  3. Or to initialise inidividually just call the function to add whichever entity you want to add to window/level editor through handles()
