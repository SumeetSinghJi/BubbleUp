\doxysubsection{Detailed Description}
Particle graphic render effect. 

You can use this in any software e.\+g, game to draw particle effects such as treading on water, lava from volcanoes, clouds, stars, or any image that you can represent with dots

EXAMPLE


\begin{DoxyEnumerate}
\item Create a vector to hold all the particles std\+::vector$<$\+Particle$>$ particles\{\};
\item In a event e.\+g. Mouse handle click pass a preconfigured animation in the 3rd parameter fireworks = random colours and transparency levels e.\+g. a firework water = Water effects e.\+g. water splashes fire = fire effects e.\+g. candle flame
\end{DoxyEnumerate}

\item In your rendering e.\+g. draw() draw the particles vector Particle\+::render\+\_\+particles(particles, renderer);
\item (Optional) Clear particles on exit depending on how you setup your camera/particles vector scope Particle\+::clear\+\_\+particles(particles, SCREEN\+\_\+\+HEIGHT); 
\end{DoxyEnumerate}
