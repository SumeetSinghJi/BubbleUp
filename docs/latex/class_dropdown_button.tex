\doxysubsection{Detailed Description}
SDL Push Button subclass of \doxylink{_base_button_8hpp_source}{Base\+Button.\+hpp} base class. 

To be used to add a drop down button that on click shows a list of all drop down buttons. You can dynamically add or remove buttons from the list. An example use case would be for a navigation bar menu e.\+g, the traditional Windows File -\/\texorpdfstring{$>$}{>} save as, windows options.
