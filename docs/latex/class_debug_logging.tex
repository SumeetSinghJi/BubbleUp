\doxysubsection{Detailed Description}
debug logging solution class to output traditional cout debug logs to terminal, buffer and file depending on severity. 

To enable a software to have a better logging solution then simply couting messages to terminal, or outputting all cout\textquotesingle{}s to a save file you can use this class to implement a complete logging solution.
